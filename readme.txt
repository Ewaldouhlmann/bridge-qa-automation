Automatização para realizar testes de software utilizado no processo seletivo da Bridge.
Requisitos:
    Python (versão 3.6 ou superior)
    Bibliotecas necessárias (listadas no requirements.txt)

Para testar o programa, seguir os seguintes passos:
1. Ativar ambiente virtual
1. instalar dependencias: pip install -r requirements.txt
2. Adicione as seguintes variáveis ao arquivo .env, substituindo seu_email@example.com e sua_senha pelas suas credenciais do sistema.
3. Rodar no terminal: Windows: python test_.py
                    Linux: python3 test_.py
